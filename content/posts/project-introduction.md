## 这个项目是什么

这是一个个人博客系统。它使用 Vue 作为访客端前台，NestJS 提供后端 API，Prisma 管理数据库模型，PostgreSQL 保存文章、分类、标签和用户等数据。

## 当前已经完成什么

目前项目已经具备最小可用的只读博客能力：

- 首页文章列表
- 文章详情页
- 归档页面
- 关于页面
- Markdown 文章导入
- Markdown 文章全量同步

## 后续计划

接下来会继续完善内容管理体验，比如更方便的文章编辑方式、后台管理、评论审核、站点配置和部署流程。目标是让这个博客既能快速上线，也能长期维护。
